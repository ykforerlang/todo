DeviceEventEmitter

RN的DeviceEventEmitter 可以实现发送/监听事件的功能，对于这个事件的这个功能， 我们提供了@areslabs/wx-eventemitter 这个包

使用方式
import {Event, eventEmitter} from '@areslabs/wx-eventemitter'

// 添加监听
const cancel = eventEmitter.on('load', (preload) => {
    console.log(preload)
})

// 取消监听
cancel()


// 触发事件
eventEmitter.emit(new Event('load', 2))

Event是一个事件类， 构造参数接受两个

  1. type: 事件类型
  2. preload: 事件携带信息

eventEmitter 有两个方法

  1. emit: 触发事件
  2. on: 添加监听， 监听函数的第一个参数是Event
